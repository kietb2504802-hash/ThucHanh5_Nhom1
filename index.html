ℹ️ Thông tin hỗ trợ

Hỗ trợ định dạng video:

  • MP4 (.mp4) - được hỗ trợ rộng rãi
  • WebM (.webm) - cho các trình duyệt hiện đại
  • Ogg (.ogv) - cho tính tương thích tốt hơn

Cách sử dụng:

  1. Đặt file infographic.jpg trong cùng thư mục với file HTML
  2. Đặt file video.mp4 trong cùng thư mục với file HTML
  3. Mở file HTML trong trình duyệt web
